Selecting withjQuery

You can use the $ sign to select the elements from the DOM! it will feel a lot like selectong using querySelectorAll().

For example:

  • Selecting all a tags:
    • $('a')
  • Selecting all elements from the class "container"
    • $(".container")
  • Selecting all elemenrs with id="special"
    • $("#special")
